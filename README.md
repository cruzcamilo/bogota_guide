# Bogota Guide

This app shows lists of recommended places to visit in Bogotá in 4 categories:

- Museums
- Parks
- Restaurants
- Malls

# Features

- Use of `Listview`, `ArrayAdapter`, `ViewPager` y `ViewHolder` pattern.
- Each place has information of: address, hours and website.
